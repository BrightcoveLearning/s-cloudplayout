Introduction

The Schedule Calendar is essentially a visual tool that helps you plan, organize, and schedule your media content in advance. Here we will see how to manage the calendar, including how to add and re-order content items.

Schedule Calendar overview

After you create your channel, click Schedule in the navigation header. Next, we will explain the features in the Schedule Calendar.

  1. Once you are in the Scheduled Calendar section, you will find the main pointers that will help you locate where and when to add your content. For example:

    1. The green division reflects the current time on the calendar timeline. This division can't be moved or selected.
    2. The yellow division reflects the selected time on the calendar timeline. This division helps you select the time when you want to add content. You can move this division as long as it is greater than the current time.

      3. Note: You can change the calendar time visibility between hours and minutes if you want more precision.

Adding content

Videos and playlists (manual or intelligent) can be added to the schedule. In addition, you can add live events. The following sections provide steps for adding each kind of content.

Note: Only videos with MP4 rendition can be added to the schedule.
Note: When adding a playlist, if a video in the playlist doesn't have an MP4 rendition, this video will not appear in the schedule.
